## 一个疑问

作者是非常不明白为什么要建设 Great FireWall [1]，因为从计算机技术和开源的角度而言，这个障碍是没有半分益处的，相反，它的害处却是十分的大，它可以让被墙的用户发展停滞，甚至是忘掉已经掌握的工程和技术。
